# API

## version 1

Prefix: /api/v1

### POST /register

Register.

### POST /login

Login.

### POST /logout

Logout.

### GET /user

Get current user.

### GET /user/{id}

Get user with id.

### POST /groups

Create a new group.

### GET /groups

List all groups the current user in.

### POST /groups/{id}

Join the groups with id.

### GET /groups/{id}

Get group details.
